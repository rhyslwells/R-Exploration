## Factors in R

Converting a numeric variable to a factor in R is crucial for **proper categorical data handling**:

**Why convert to factor:**

1. **Semantic clarity** — Even though maturity stages are coded as 1-5 (numbers), they represent *categories*, not a continuous scale. A factor tells R these are discrete groups, not quantities.

2. **Statistical modeling** — When you use a factor in functions like `lm()`, `aov()`, or `glm()`, R automatically treats it as categorical and creates dummy variables. Without the factor conversion, R would treat it as a continuous numeric predictor, which is biologically incorrect.

   ```r
   # Treats maturity.stage as numeric (WRONG)
   lm(weight ~ maturity.stage, data = squid)
   
   # Treats Fmaturity as categorical (CORRECT)
   lm(weight ~ Fmaturity, data = squid)
   ```

3. **Group comparisons** — ANOVA, Tukey tests, and other group comparison tests assume the grouping variable is a factor.

4. **Visualization** — Functions like `boxplot()` and `plot()` handle factors more intelligently, automatically treating each level as a separate group.

5. **Prevents accidental calculations** — Without factorization, you could accidentally compute `mean(squid$maturity.stage)` (treating stages as numbers), which is nonsensical biologically.

6. **Ordered levels** — Factors let you control the order of categories in plots and summaries, preventing alphabetical or arbitrary sorting.

**In your code**, converting `maturity.stage → Fmaturity` as a factor ensures all downstream analyses treat reproductive maturity as the categorical grouping variable it actually is.

## Testing for normality and variences

what assumption does the t-test make about the data? Normal distribution and equal variances

what does the t test do: compares the means of two groups and tests whether they are statistically significantly different from each other.

looking to say that its not normal

In R

```r
# Do not perform test on all data together, i.e.

shapiro.test(prawns$GRate) # this is wrong!!

# Need to test for departures from normality for each group 
# separately. Remember your indexing [ ]

shapiro.test(prawns$GRate[prawns$diet == "Artificial"])
 
#         Shapiro-Wilk normality test
# 
# data:  prawns$GRate[prawns$diet == "Artificial"] 
# W = 0.9486, p-value = 0.1553

shapiro.test(prawns$GRate[prawns$diet == "Natural"])

#         Shapiro-Wilk normality test
# 
# data:  prawns$GRate[prawns$diet == "Natural"] 
# W = 0.9598, p-value = 0.307

# Therefore no evidence to reject the Null hypothesis and data are normally distributed

# However much better assessment of normality is to use a quantile - quantile plot
# looking for points to lie along the line for normality
```
